Correct first turn in operation
===============================

This template corrects a first-turn trajectory using an operational
``InterfaceInjection`` implementation, a saved one-turn trajectory response
matrix, and an optional name mapping from model names to control-system names.

Download the script: :download:`correct_1st_turn.py <../../../../examples/operation/correct_1st_turn.py>`
