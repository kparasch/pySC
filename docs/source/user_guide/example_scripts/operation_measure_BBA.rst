Measure BBA in operation
========================

This template measures beam-based alignment for one BPM with an operational
``Interface``. It loads a BBA configuration, runs ``measure_bba``, and analyzes
the horizontal and vertical results when each plane is done.

Download the script: :download:`measure_BBA.py <../../../../examples/operation/measure_BBA.py>`
