Measure ORM in operation
========================

This template measures an orbit response matrix with an operational
``Interface``. It iterates over the ``measure_ORM`` generator, logs progress,
and shows how to optionally correct the orbit after each corrector measurement.

Download the script: :download:`measure_ORM.py <../../../../examples/operation/measure_ORM.py>`
